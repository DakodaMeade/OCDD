							Automotive Detailing Web Application
Overview
This web application is designed for an automotive detailing company, providing customers with an intuitive interface to book detailing services, view their appointments, 
and leave reviews. The admin dashboard allows management of services, appointments, and customer interactions.

All final project artifacts (Project Proposal, Project Requirments, Project Architecture, Project Development, Data Dictionary, Test Case, Tracibility Matrix) are included. 


Features
	User Registration and Login: Users can create an account, log in, and manage their profiles.
	Appointment Scheduling: Customers can schedule, view, and manage their appointments online.
	Profile Management: Users can mamange their profile.
	Appointments Managment: Allows users to view and manage their appointments.
	Service Management: Admins can add, edit, or remove detailing services offered.
	Review System: Customers can leave reviews about the services they've received.
	Admin Dashboard: A comprehensive dashboard for managing appointments, services, and customer reviews.
	Technologies Used
	Frontend: HTML, CSS, JavaScript, Bootstrap
	Backend: ASP.NET Core MVC, C#
	Database: Azure MYSQL
	Cloud Deployment: Azure

Getting Started

Prerequisites
	.NET SDK
	Visual Studio
	MYSQL Server or Azure MYSQL Database


GitHub Repository: https://github.com/DakodaMeade/OCDD

Installation (Copy and run locally)
Clone the repository:

	git clone https://github.com/DakodaMeade/OCDD

Navigate to the project directory:

	cd OCDD

Restore dependencies and build the project:

	dotnet restore
	dotnet build

Update the database connection string in appsettings.json to point to your database.

Run the application:

	dotnet run

Access the application in your browser at https://localhost:[port]. (Enter the port it is running at)

Deployment
